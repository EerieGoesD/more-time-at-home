# More Time at Home
Optimize paid leave to maximize total days off around weekends and national holidays.

## Features
- Automatic public holiday loading (Nager.Date API)
- Leave optimization engine
- Min/Max consecutive day constraints
- Full-year calendar view
- Save/Load plans
- Runs locally (no accounts, no tracking)

## Tech
- .NET 6 WPF
- C#
