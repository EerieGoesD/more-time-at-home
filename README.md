# More Time at Home

A cross-platform desktop app that helps you maximize your time off. Enter your country, available paid leave days, and any constraints — the optimizer finds the smartest way to distribute your leave around national holidays and weekends.

- **Smart optimizer** — Automatically places paid leave days where they unlock the most consecutive time off, including bridge days (single workdays between holidays/weekends).
- **Holiday calendar** — Loads national holidays for any country via the [Nager.Date API](https://date.nager.at). Manually add custom holidays too.
- **Constraints** — Set minimum/maximum block lengths for days off or paid leave in a row.
- **ICS export** — Export your plan to `.ics` files for Google Calendar, Outlook, Apple Calendar, etc.
- **Desktop notifications** — Configurable reminders before each planned leave block.

Built with [Tauri](https://tauri.app), React, TypeScript, and Vite.
